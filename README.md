**Forecast Accuracy & Bias Dashboard using Power BI
**Overview

This project presents an interactive Power BI dashboard for analyzing forecast performance against actual values. It enables users to compare multiple forecast snapshots (Resultant, Lag0, and Lag1) with actual data and evaluate forecasting quality using Forecast Accuracy and Forecast Bias KPIs.

Features
Data transformation using Power Query
Pivoting and unpivoting of forecast data
Merging Forecast and Actual datasets
Interactive Lag Selector (Resultant, Lag0, Lag1)
Dynamic DAX measures
Forecast Accuracy and Forecast Bias calculations
Country-wise Revenue, Non-Revenue, and CAPEX analysis
Executive dashboard with interactive visualizations
Tools Used
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Data Processing
Imported Forecast and Actual datasets.
Cleaned and transformed the data using Power Query.
Created Lag labels (Resultant, Lag0, Lag1).
Unpivoted and pivoted forecast measures.
Merged Actual and Forecast datasets.
Created dynamic measures using DAX.
Built interactive dashboard visuals and KPI cards.
Dashboard Insights
Compare forecast values across different forecast snapshots.
Measure forecasting accuracy using KPI metrics.
Analyze forecast bias.
Compare Revenue, Non-Revenue, and CAPEX across countries.
Interactive filtering using the Lag Selector.
Outcome

The dashboard provides a clear view of forecast performance, enabling users to identify forecasting errors, compare historical forecast versions, and support better business decision-making.
